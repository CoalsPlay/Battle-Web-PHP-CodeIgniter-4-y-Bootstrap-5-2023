[English]

In this same directory it includes an HTML file where you can see a more complete documentation on all Battle Web PHP,
from its installation to each of its integrated systems.

The product does not have the images of enemies/store items/maps used in the demo for possible Copyright reasons.
To use your own images you must deposit them in "/assets/img/items" for the store, "/assets/img/maps" for the maps
and "/assets/img/mobs/" for enemies. Check the documentation for more details!

[Spanish]

En este mismo directorio incluyo un archivo HTML donde podrás ver una documentación más completa sobre todo Battle Web PHP,
desde su instalación hasta cada uno de sus sistemas integrados.

El producto no cuenta con las imágenes de enemigos/productos de tienda/mapas utilizados en la demo por cuestiones de posible Copyright.
Para utilizar sus propias imágenes deben depositarlas en "/assets/img/items" para la tienda, "/assets/img/maps" para los mapas
y "/assets/img/mobs/" para los enemigos. ¡Revise la documentación para más detalles!
